[0053: Search read multiple vss branches with a single path using wildcard]

Test assertion:
When the client makes getRequest to the server with filter containing "paths" type and correct value,
a JSON data object that contain timestamp and multiple vss data will return as response.
In this case, value is single vss path pointing vss branch with wildcard.

Test Scenario:
1. Send getRequest specifying an action, data path, requestId and filter.
2. Confirm response containing the specified action, requestId, data from correct path that satisfy the given filter.

Expected Result:
- Received 'getSuccessResponse' containing specified requestId.
